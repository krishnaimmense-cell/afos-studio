# UI / UX DESIGN

Version: 1.0

---

# Purpose

This document defines the complete user experience for AFOS Studio.

The goal is to make professional filmmaking accessible while maintaining a premium, cinematic experience.

---

# Design Philosophy

AFOS is not a chatbot.

AFOS is professional filmmaking software.

Users should feel like they are operating a film studio rather than talking to an AI.

---

# Core Principles

- Mobile First
- Simple Navigation
- Professional Appearance
- Guided Workflow
- Minimal User Decisions
- Fast Production
